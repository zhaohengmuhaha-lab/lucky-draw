# 幸运大转盘抽奖系统

这是一个基于纯 HTML/CSS/JavaScript 开发的轻量级转盘抽奖系统，专为活动运营和个人用户设计。无需后端、无需配置，打开浏览器即可使用。

## 🌟 核心功能

*   **自定义奖品**：支持设置 5 个奖品名称。
*   **自定义概率**：可以为每个奖品单独设置中奖概率（总和需 ≤ 100%）。
*   **自适应布局**：完美适配 PC 端（左右布局）和移动端（上下布局）。
*   **平滑动画**：模拟真实的加速、匀速、减速转盘效果。
*   **所见即所得**：单文件运行，无任何外部依赖。

## 🚀 如何使用

### 方法一：直接运行
1.  下载本项目中的 `lottery_system.zip` 或 `index.html`。
2.  双击 `index.html` 在浏览器中打开。
3.  在页面输入框中填写奖品名称和对应的概率。
4.  点击“开始抽奖”！

### 方法二：在线部署
本系统包含 `index.html`，可以直接上传至 GitHub Pages、Vercel 或 Netlify 进行免费托管。

## ⚙️ 参数配置

在页面中直接输入即可配置：
*   **奖品名称**：输入 5 个奖品的名字。
*   **中奖概率**：为每个奖品输入 0-100 的整数，系统会实时校验总概率。

## 🛠️ 二次开发

如果你懂一点代码，可以右键用记事本或编辑器打开 `index.html` 进行修改：
*   **修改配色**：搜索 `COLORS` 数组修改扇形颜色。
*   **调整时长**：搜索 `duration` 变量修改转动时间（默认 4秒）。
